Code Anywhere notes for Kotlin

1. Choose: Container

OS: Ubuntu 16.04

Stack: Java

This installs Java 1.8.0_181

2. Install SDKMAN

curl -s https://get.sdkman.io | bash

3. Add SDKMAN to path

source "/home/cabox/.sdkman/bin/sdkman-init.sh"

4. Install Kotlin

sdk install kotlin

5. Verify Kotlin compiler was installed

kotlinc -version

(I got 1.3.21 for JRE 1.89.0_181_b13 when I did this March 11, 2019)
